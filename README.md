# LangChain Demos

Pequeña colección de demos con LangChain y OpenAI para mostrar en tu portafolio.

Contenido:
- explain_topic: demo que genera explicaciones breves sobre un tema. (explain_topic.py)
- interactive_chat: chat CLI con historial (interactive_chat.py)
- math_agent_demo: agente que usa la herramienta de matemáticas (math_agent_demo.py)

Requisitos

```bash
python -m venv .env
.env\Scripts\activate
pip install -r requirements.txt
set OPENAI_API_KEY=tu_clave_de_openai
```

Ejecución de ejemplos

- Explicador:

```bash
python explain_topic.py
```

- Chat interactivo:

```bash
python interactive_chat.py
```

- Demo agente matemático:

```bash
python math_agent_demo.py
```
