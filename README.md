# Research Agent (LLM-based)

This project is a small research assistant that uses language models and web tools to gather information and produce a structured response.

## What it does
- Runs an LLM-based agent that can call simple tools (web search, Wikipedia, save-to-file) to answer a user's research query.
- Parses the agent output into a Pydantic model so the result can be used programmatically.

## Requirements
Install the Python dependencies (preferably inside a virtual environment):

```bash
python -m venv venv
venv\Scripts\activate   # Windows
pip install -r requirements.txt
```

## Configuration
- Create a `.env` file in the project root with any API keys required by your LLM providers (OpenAI, Anthropic, etc.). Example:

```
OPENAI_API_KEY=your_openai_key
ANTHROPIC_API_KEY=your_anthropic_key
```

## How to run
Run the script:

```bash
python main.py
```

The script contains an example invocation of the agent executor and prints the raw response. You can switch to interactive input by uncommenting the `input()` example in `main.py`.
